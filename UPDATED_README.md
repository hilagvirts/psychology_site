## 🎯 ההבדלים בין הסגנונות

### 🆚 לפני (Original) vs אחרי (Bar-Ilan)

| אלמנט | לפני | אחרי (Bar-Ilan) |
|-------|------|-----------------|
| **רקע** | גרדיאנטים צבעוניים | רקע לבן נקי |
| **צללים** | כבדים ובולטים | עדינים מאוד או גבולות |
| **צבעים** | צבעוניים ועזים | כחול אקדמי עדין |
| **אנימציות** | bounce, pulse, fade | fade בלבד |
| **כותרות** | 700 (Bold) | 300-500 (Light-Medium) |
| **מרווחים** | סטנדרטיים | נדיבים מאוד |
| **גבולות** | עבים (3-4px) | דקיקים (1px) |
| **Hero** | גרדיאנט + תבניות | גרדיאנט פשוט |
| **Cards** | צללים כבדים | גבולות פשוטים |
