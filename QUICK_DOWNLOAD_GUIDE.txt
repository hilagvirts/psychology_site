שלב 2: הורד 6 קבצי HTML לתיקייה הראשית
═══════════════════════════════════════════════════════════════════════════

📄 index.html                      → psychology-website/index.html
📄 clinical-english.html           → psychology-website/clinical-english.html
📄 organizational.html             → psychology-website/organizational.html
📄 rehabilitation-adults.html      → psychology-website/rehabilitation-adults.html
📄 rehabilitation-children.html    → psychology-website/rehabilitation-children.html
📄 neuroscience.html               → psychology-website/neuroscience.html

איך להוריד קובץ HTML:
1. פתח את הקובץ בפלטפורמה
2. Ctrl+A (בחר הכל)
3. Ctrl+C (העתק)
4. פתח Notepad
5. Ctrl+V (הדבק)
6. File → Save As
7. שמור עם השם המדויק (כולל .html)
