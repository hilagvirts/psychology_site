📝 הוראות הורדה - שלב אחר שלב
═══════════════════════════════════════════════════════════════════════════

שלב 1: צרי תיקייה על המחשב
─────────────────────────────
1. פתחי את סייר הקבצים (File Explorer)
2. נווטי לשולחן העבודה (Desktop)
3. צרי תיקייה חדשה בשם: psychology-website


שלב 2: הורידי כל קובץ HTML
──────────────────────────
לכל אחד מ-6 הקבצים, עשי:

1. פתחי את הקובץ DOWNLOAD_[שם].html.txt בפלטפורמה
   
2. גללי למטה עד שתראי את השורה:
   "─── התחל להעתיק מכאן ───"

3. בחרי את כל התוכן מתחת לשורה הזו (מ-<!DOCTYPE עד </html>)
   טיפ: לחצי בתחילת <!DOCTYPE ואז Ctrl+Shift+End

4. העתיקי (Ctrl+C)

5. פתחי Notepad במחשב שלך
   (Start → חפשי "Notepad")

6. הדביקי (Ctrl+V)

7. לחצי: File → Save As

8. הגדרות שמירה:
   • Save in: psychology-website (התיקייה שיצרת)
   • File name: השם המדויק (למשל: index.html)
   • Save as type: All Files (*)
   • Encoding: UTF-8
   
9. לחצי Save

10. חזרי על השלבים 1-9 לכל 6 הקבצים
